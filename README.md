
This project demonstrates a complete DevOps CI/CD pipeline on AWS using Terraform, Jenkins, Docker, Amazon ECR, Amazon EKS, Prometheus, and Grafana.
The pipeline automatically builds, stores, and deploys a containerized Node.js application whenever code is pushed to GitHub. 
# flow :
GitHub → Jenkins → Docker → Amazon ECR → Amazon EKS → Prometheus → Grafana

# Used :
* AWS EC2 * AWS ECR * AWS EKS * Terraform * Jenkins * Docker * Kubernetes * GitHub Webhooks * Prometheus * Grafana * Node.js
